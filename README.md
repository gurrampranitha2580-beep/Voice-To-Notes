# Introduction
During lectures, it is often difficult for students to listen and take notes at the same time. Important points can be missed.  
This project aims to make studying easier by converting lecture audio into text notes automatically.

# Features
- Upload lecture audio (WAV format)
- Automatically transcribes speech into text
- Simple and user-friendly interface
- Free and works on any computer
